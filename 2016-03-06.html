Firefox へのバグ報告

"Bug" として報告するもの

Firefox のバグ・パッチ・要望などはすべて "Bug" として Bugzilla に集約される。

  • セキュリティバグだけじゃない
  • XPS で印刷するとクラッシュ
  • Flexbox のレンダリング結果がおかしい
  • 日本語入力ができない
  • これらは実際に報告される "Bug"

DevTools Reload

DevTools Reload

2/24 にブログ発表された Mozilla 製アドオン

  • Firefox Developer Tool をハックしやすく
  • Web 開発者もコントリビュートしやすく
  • Ctrl + Alt + R 一発でコード変更を反映
  • ビルド環境を整える必要がない

使ってみる

MDN の記事に詳しく載ってる

  1. Firefox Nightly をインストール
  2. git clone --depth 1 で shallow clone
  3. アドオンを Nightly にインストール
  4. ソースコードをいじる (XML/CSS/JS)
  5. Ctrl + Alt + R でリロード
  6. 手順 4 - 5 を繰り返す

知っておくとよいこと

devtools/client 配下がいじりやすい

  • chrome://devtools/content/foo は
    devtools/client/foo に解決される
  • chrome://devtools/skin/foo は
    devtools/client/themes/foo に解決される
  • Browser Toolbox はいいぞ
